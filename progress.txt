## Architecture Decisions
- Runtime: IronClaw (Rust, WASM sandboxing, MCP protocol, PostgreSQL, identity files)
- LLM: Grok-4.20 via xAI OpenAI-compatible API (https://api.x.ai/v1)
- Knowledge graph: Neo4j 5.26+ community edition in Docker
- Embeddings: xAI embedding endpoint, 1536 dimensions, cosine similarity
- File watcher: fsnotify (FSEvents on macOS) with 60-second polling fallback
- Large note chunking: 2000-token sliding window, 200-token overlap, deduplicate >0.95 cosine
- Docker containers: all use --restart unless-stopped
- Obsidian vault is canonical data store — if Neo4j disagrees, Obsidian wins
- Lingelpedia Agent never deletes knowledge in operational mode
- All migrations must be idempotent
